## API

| 属性     | 说明         | 类型             | 默认值 |
| -------- | ------------ | ---------------- | ------ |
| nodeSize | 节点的直径   | number           | 60     |
| data     | 关系图的数据 | object           | -      |
| width    | 宽度         | number 或 string | 100%   |
| height   | 高度         | number 或 string | 500    |
