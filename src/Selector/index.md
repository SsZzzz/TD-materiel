## API

| 属性             | 说明                   | 类型   | 默认值 |
| ---------------- | ---------------------- | ------ | ------ |
| className        | 宽高可以在这里设置     | string | -      |
| contentClassName | 布局样式可以在这里设置 | string | -      |
| onSelect         | 框选的回调事件         |        | -      |
| selection        | 是否可框选             |        | true   |
