## DisableRangePicker

基于 antd 的 rangePicker,增加时间范围的限制,方便给 form 表单使用.

## 示例

<code src="./demo.jsx"></code>

## API

| 属性      | 说明                       | 类型   | 默认值 |
| --------- | -------------------------- | ------ | ------ |
| diffType  | years、months、weeks、days | string | days   |
| diffCount | 时间范围                   | number | 6      |

其他参数与 antd 的 rangePiker 一致
